# BurnWatch 🔥

**BurnWatch** is a personalized burnout prevention engine designed to help users monitor mental and physical strain. By combining clinical assessments with daily biometric and lifestyle logging, it provides a data-driven "BurnScore" and visualizes burnout trends.

---

##  Features

* **User Authentication**: Persistent local session management using `localStorage`.
* **OLBI Baseline Assessment**: Scientific burnout risk categorization based on the Oldenburg Burnout Inventory.
* **Daily Log Wizard**: Interactive step-by-step logging of physical activity (steps, HR), rest (sleep, screen time), and mental state.
* **Dynamic BurnGraph**: Real-time data visualization using **Chart.js**, featuring a 7-day trend prediction based on linear regression.
* **Panic Button**: Instant recovery interventions and tips for immediate stress relief.
* **Glassmorphism UI**: A modern, dark-themed interface utilizing CSS gradients and backdrop filters.

---

##  Tech Stack

* **Frontend**: HTML5, CSS3 (Custom properties, Animations)
* **Logic**: Vanilla JavaScript (ES6+)
* **Data Visualization**: Chart.js

---

##  Installation & Usage

To run this project locally, follow these steps:

1.  **Download** all repository files into a single directory.
2.  Ensure the file structure remains intact (all `.html`, `.css`, and `.js` files in the same root folder).
3.  **Launch** the application by opening `login.html` in any modern web browser.
